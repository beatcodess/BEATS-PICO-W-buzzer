To get started 
1. get a pico wh ( i used a pico w i regret it)
2. get a buzzer and 2 female to female jumpers (i used male to female but mine didt have pins)
3. connect the jumpers to pins GP 15 and GRD
4. download thonny IDE https://thonny.org/ 
5. downlaod micro python off of it
6. copy my code into it
7. edit it how you would like
8. save the file as main.py to the pico wh 
plug and play plug it in anywhere as long as your ssid (network name) and wifi password are right for the wifi your using your good and access the ip via phone tablet laptop etc.
